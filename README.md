# Order Rush

เกม Time-Management & Action-Puzzle (คล้าย Overcooked) ผู้เล่นรับบทพนักงานคลังสินค้า: รับออเดอร์ → หยิบของ → แพ็ก → ส่ง ภายในเวลาที่จำกัด

เล่นได้ในเบราว์เซอร์ทันที ไม่ต้องติดตั้งอะไร — ไฟล์เกมทั้งหมดอยู่ใน [`index.html`](index.html) ไฟล์เดียว (HTML + CSS + JS inline) ใช้ [Phaser 3](https://phaser.io/) ผ่าน CDN วาดกราฟิกทั้งหมดด้วย Graphics API (ไม่มีไฟล์รูปภาพ) และใช้ Web Audio API สร้างเสียงเอง (ไม่มีไฟล์เสียง)

## วิธีเล่น

1. แตะสินค้าด้านล่างจอเพื่อแพ็กลงกล่อง
2. แพ็กของให้ตรงกับออเดอร์ที่เลือกไว้ (การ์ดหลักตรงกลางจอ)
3. กดปุ่ม "ปิด & ส่ง! 🚀" ตอนของในกล่องครบ
4. ส่งถูกต่อกันสร้างคอมโบ ยิ่งคอมโบสูงยิ่งได้โบนัสคะแนนตอนจบเกม

## วิธีรันเล่นในเครื่อง

เปิดไฟล์ `index.html` ตรงๆ ในเบราว์เซอร์ได้เลย หรือใช้ static server เล็กๆ ที่เตรียมไว้ให้ (สำหรับทดสอบบนมือถือในวง LAN เดียวกัน):

```powershell
./serve.ps1
```

ค่าเริ่มต้นจะรันที่พอร์ต `7874`

## โครงสร้างไฟล์

- `index.html` — เกมทั้งหมด (Phaser 3 Scene, UI, ระบบเสียง, ระบบคะแนน/คอมโบ, localStorage high score)
- `serve.ps1` — static file server เล็กๆ (PowerShell, TCP raw) สำหรับทดสอบบนอุปกรณ์อื่นในวง LAN เดียวกัน
- `PROMPT.md` — Game Design Document ต้นฉบับที่ใช้เริ่มสร้างเกมนี้

## ข้อจำกัดของโปรเจกต์

- ไฟล์เดียว ไม่มี build step, ไม่มี backend, ไม่มีระบบออนไลน์/เติมเงิน
- ไม่ใช้ไฟล์รูปภาพ/เสียง ใช้ Graphics API และ Web Audio API ล้วนๆ
